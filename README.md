# Build
## Source

    $ git clone https://github.com/neo1218/neo1218_blog.git
    $ cd neo1218_blog
    $ chmod 777 pull.sh
    $ ./pull.sh

## Virtual Environment

    $ Virtualenv venv
    $ source venv/bin/activate

## Install Extensions

    $ pip install -r requirment.txt

## Run

    $ python manage.py runserver
