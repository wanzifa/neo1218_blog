## 指针是什么
指针是一个变量,占用内存,指针的值就是其指向的地址。

## 引用是什么
引用就是一个对象的别名。其实这个解释只是语言在应用层面的给使用语言的人的解释,引用实际上就是一个常量指针。具体可以参考[这篇文章](http://www.cnblogs.com/ronny/p/3662556.html),
以及我做的这个[小实验]()。

## 举个例子🌰
有如下代码

    int main()
    {
        int i = 2;
